Responsive Piano Project

This project is a fully responsive piano interface built with HTML & CSS.
The main focus of this project was making a traditionally wide layout scale smoothly across different screen sizes.

What I Built

- A piano container with rounded edges

- White and black piano keys created using pure CSS

- A floating black-key effect using pseudo-elements

- A positioned logo inside the piano frame
